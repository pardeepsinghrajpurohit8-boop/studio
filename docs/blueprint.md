# **App Name**: CalcPrice

## Core Features:

- Quantity Input: Allow the user to input the quantity of the item.
- Price Input: Allow the user to input the price of the item.
- Price Calculation: Calculate the total price by multiplying quantity and rate.
- Realistic Interface: Render buttons and views with a realistic 3D look and feel.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey a sense of technological advancement.
- Background color: Light gray (#E0E0E0) to provide a neutral and clean backdrop.
- Accent color: Bright cyan (#00FFFF) for highlighting interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral look.
- Use minimalist, futuristic icons to represent functions and interactive elements.
- Arrange UI elements with a clean and efficient layout for easy data entry and viewing of results.
- Subtle animations, such as a brief glow or pulse, when users interact with input fields or buttons.
